**The required operating environment: .NET 8.0**

## Features

The following files are supported:

| Wallpaper type | File type                       |
| -------------- | ------------------------------- |
| Motion Picture | .gif .apng                      |
| Still image    | .jpg .jpeg .bmp .png .svg .webp |
| Video          | .mp4 .webm                      |
