## Features

**Supported file types**

| Wallpaper Type  | Extensions                                          |
| --------------- | --------------------------------------------------- |
| Still Image     | `.jpg` `.jpeg` `.bmp` `.png` `.svg` `.webp`         |
| Motion Picture  | `.gif` `.apng`                                      |
| Video           | `.mp4` `.webm`                                      |
| Web Interactive | `.zip` `.rar` `.7z`                                 |
